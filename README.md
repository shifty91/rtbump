# rtbump #

## About ##

My personal tooling for keeping the rt-sources in Gentoo update to date.

## Usage ##

    usage: rtbump.pl [options]

    options:
        --branch,  -b: The branch created
        --dry_run, -d: Dry run
        --help,    -h: Show this help
